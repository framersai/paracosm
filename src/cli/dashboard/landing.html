Core Capabilities

What makes Paracosm different

Not a chatbot wrapper. A simulation engine where AI agents invent tools, evolve personalities, and generate crises nobody programmed.

Runtime Tool Forging

Department agents create computational tools on the fly: radiation calculators, food projectors, morale models. Each tool is sandboxed in V8 and reviewed by an LLM-as-judge for safety and correctness.

Built on AgentOS EmergentCapabilityEngine + EmergentJudge

HEXACO Personality Drift

Six-factor personality model from psychology research. Continuous 0-1 traits drift through leader pull, role activation, and outcome reinforcement across turns. Same person, two commanders, measurably different behavior.

Ashton, M.C. & Lee, K. (2007). Personality and Social Psychology Review, 11(2), 150-166

Emergent Crisis Director

An LLM agent observes colony state, resource levels, decision history, and tool intelligence. It generates unique crises per timeline that exploit consequences of prior decisions and escalate over time.

Adaptive narrative generation, cf. Riedl & Bulitko (2013)

Deterministic Kernel

Seeded PRNG (Mulberry32) ensures reproducibility. Same seed produces the same colonist roster, births, deaths, and promotions. Divergence comes entirely from AI-driven decisions.

Law, A.M. & Kelton, W.D. (2000). Simulation Modeling and Analysis. McGraw-Hill

Scenario Compiler

Define any closed-state settlement as JSON. The compiler generates all runtime hooks via LLM: progression logic, crisis templates, milestone conditions, political dynamics. ~$0.10 per compile, cached to disk.

Zero-code scenario authoring via AgentOS generateText()

Research-Backed Decisions

Department agents cite DOI-linked academic papers. Decisions are grounded in domain literature via semantic search across Serper, Tavily, Firecrawl, and Brave with Cohere neural reranking.

Cohere Rerank v3.5 for citation relevance scoring

How a turn works

Each turn is a complete cycle. Crisis generation, multi-department analysis with tool forging, commander decision, deterministic state update, population reaction.

FAQ

Frequently asked questions

What is Paracosm?
A scenario-driven multi-agent simulation engine. You define a ScenarioPackage describing your world, and the engine runs it with emergent AI crisis generation, multi-agent analysis, runtime tool forging, HEXACO personality drift, and deterministic state transitions.
How does tool forging work?
Department agents create computational tools at runtime. Each tool is sandboxed in an isolated V8 environment, reviewed by an LLM-as-judge for safety and correctness, and produces real computed output that influences decisions. Nobody pre-programmed these tools.
AgentOS EmergentCapabilityEngine + EmergentJudge
What is HEXACO personality?
A six-factor personality model: Honesty-Humility, Emotionality, Extraversion, Agreeableness, Conscientiousness, and Openness to Experience. Each trait is a continuous 0-1 value that drifts over time through leader pull, role activation, and outcome reinforcement.
Ashton, M.C. & Lee, K. (2007). "Empirical, Theoretical, and Practical Advantages of the HEXACO Model." Personality and Social Psychology Review, 11(2), 150-166.
Is the simulation deterministic?
The kernel is fully deterministic. Same seed produces the same colonist roster, births, deaths, and promotions via Mulberry32 PRNG. Divergence comes entirely from AI-driven decisions.
Law, A.M. & Kelton, W.D. (2000). "Simulation Modeling and Analysis." McGraw-Hill.
What LLM providers are supported?
OpenAI (GPT-5.4, GPT-5.4-mini) and Anthropic (Claude Sonnet 4.6, Claude Haiku 4.5). Different models can be assigned to different roles: commander, departments, judge, and crisis director.
Can I create my own scenario?
Two ways. Write a ScenarioPackage in TypeScript with full control over hooks. Or use the Scenario Compiler, which generates all runtime hooks from a JSON world definition via LLM (~$0.10 per compile, cached to disk).
What verticals does this support?
Defense and intelligence (wargaming), corporate strategy (leadership modeling), game studios (procedural civilizations), academic research (AI decision experiments), government (policy simulation). Any domain where testing decisions before making them has value.
Is this open source or commercial?
Both. The core engine is open source under Apache-2.0. The hosted and enterprise layers are the commercial roadmap: auth, persistence, exports, orchestration, private deployment, and white-label packaging built on top of the open core.
