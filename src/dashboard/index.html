What Is This?

Mars Genesis is a live multi-agent simulation. Two AI commanders lead the same colony of 100 colonists through crises spanning 50 years on Mars. Their HEXACO personality profiles drive different decisions, different tool inventions, and different civilizational outcomes. Same seed, same starting colony. Only leadership differs.

How It Works

  • Deterministic kernel handles colonist generation, aging, births, deaths, career progression, and resource accounting via seeded RNG.
  • Department agents (Medical, Engineering, Agriculture, Psychology, Governance) analyze each crisis using real Mars science with DOI-linked citations.
  • Tool forging lets agents invent computational tools at runtime (radiation models, food projectors). An LLM judge reviews each tool for safety and correctness.
  • Personality drift evolves promoted colonists' HEXACO traits over time via leader pull, role pull, and outcome feedback.
  • Commanders decide which policies to enact. Personality shapes risk tolerance and strategy.

What Do The Numbers Mean?

  • Pop = alive colonists (starts at 100). Morale = colony wellbeing 0-100%. Food = months of reserves. Deaths = cumulative.
  • Tools Forged = computational tools invented by department agents during the run.
  • HEXACO bars = O(penness) C(onscientiousness) E(xtraversion) A(greeableness) H(onesty-Humility).
  • Personality Drift = how colonist traits shifted this turn from leadership influence and crisis outcomes.
