    Tool intelligence factor. Forging computational tools is a tradeoff: newToolsThisEvent tools forged THIS event (consume time/compute, rejection risk, but enable insight) reuseCountThisEvent invocations of previously-forged tools (cheap, pure upside — institutional knowledge) forgeFailures failed forge attempts this event (judge rejected — wasted department effort) totalToolsForRun cumulative unique tools over the run (innovation index — diminishing returns)
