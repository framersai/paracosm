    OptionalactorNoun

    actorNoun?: string

    Singular display word for the swappable decision-making entity that runs each parallel counterfactual. Defaults to "actor" — the universal abstract type. Scenarios specialize it: Mars Genesis sets "commander", a hurricane scenario sets "incident commander", an AI release sets "release director", a quantum-game scenario sets "player". The engine type stays ActorConfig for SDK back-compat; this label is for UI / copy / button text rendering only.
